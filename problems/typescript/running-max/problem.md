# Running maximum

Provide a named export `runningMax(nums: number[]): number[]` that returns an
array of the same length where each element is the maximum of all elements seen
so far (a running / prefix maximum).

```
runningMax([1, 3, 2, 5, 4])  ->  [1, 3, 3, 5, 5]
runningMax([])               ->  []
```
